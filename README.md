# 📱 Swift로 간단한 앱 개발하기

> **모바일 프로그래밍 수업 프로젝트**로, Swift를 활용하여 다양한 앱 기능을 실습하고  
> 그 결과를 바탕으로 나만의 앱을 구현한 프로젝트입니다.


## 👨‍🏫 프로젝트 소개

<img src="https://github.com/user-attachments/assets/4e716fda-68ff-4932-9109-3d7115a3c677" width="450"/>

Swift의 기본 기능들을 활용하여 여러 가지 앱 기능을 단계별로 구현하고,  
최종적으로 각자 하나의 앱을 완성하는 실습 중심의 프로젝트입니다.


## ⏰ 개발 기간

- **2024년 03월 07일 ~ 2024년 06월 16일**
